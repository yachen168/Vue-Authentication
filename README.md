# vue 權限管理
一共串接 8 支 RESTful API，實現會員註冊、登入、登出、修改會員資訊、上傳檔案等功能。

## 成果 Demo
Github page: https://yachen168.github.io/

## 使用技術
- Vue CLI
- Vue Router
- Vant
- Axios

## 實現功能
- 會員註冊
- 會員登入
- 會員登出
- 忘記密碼 
- 編輯會員資訊
- 上傳個人頭像
